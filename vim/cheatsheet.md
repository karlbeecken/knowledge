# Vim cheatsheet
## Basics
- `:w` – save
- `:q` – quit
- `:wq` – save and quit
- `:h[elp] keyword` – show help for keyword

## Insert mode
- `i` – insert before cursor
- `I` – insert at the beginning of the line
- `a` – insert after the cursor
- `A` – insert at the end of the line
- `[Esc]` – exit insert mode
